# Bilboard

### An agentic AI application that generates analytical dashboards from a user prompt.

Bilboard uses a multi-agent system to create dashboard structures in JSON format and render them through the Blazor Interface Library (BIL). It is designed to generate and display software components efficiently at runtime while keeping resource usage low.

Built with .NET, Blazor, BIL, Microsoft Agent Framework, and Azure, Bilboard is fully open-source under the MIT licence.
